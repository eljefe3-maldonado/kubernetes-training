# Module 09: Detection Engineering And Threat Hunting

## Why This Matters

Many Kubernetes environments collect logs without turning them into meaningful
security signal. Detection engineering closes that gap.

## You Will Learn

- which telemetry sources matter most
- how to write detections tied to attacker behavior
- how to balance signal, noise, and response value

## Key Questions

- What events are noisy versus operationally meaningful?
- Which detections justify paging someone?
- How do detections connect to concrete response playbooks?
