# Module 08: Cluster Hardening And Platform Baselines

## Why This Matters

Platform security depends on more than workload policy. Node posture, API
surface, audit settings, and provider responsibility boundaries matter.

## You Will Learn

- cluster hardening baseline design
- node and control-plane security priorities
- managed versus self-managed responsibility splits

## Key Questions

- Which hardening actions reduce the most risk fastest?
- Which controls are owned by the provider versus the customer?
- How do audit settings support both detection and forensics?
