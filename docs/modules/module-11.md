# Module 11: Governance And Multi-Tenancy

## Why This Matters

Technical controls break down when tenancy boundaries, team responsibilities,
and platform standards are unclear.

## You Will Learn

- multi-tenant risk tradeoffs
- platform guardrail design
- operating model design for platform and security teams

## Key Questions

- When is a shared cluster acceptable?
- Which controls should be standardized centrally?
- How do you preserve separation of duties without slowing delivery?
