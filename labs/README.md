# Labs Index

This directory contains the hands-on portion of the Kubernetes Security Master
Class. Each module includes a lab guide and, where relevant, supporting YAML or
worksheet files.

## Modules

- [Module 01](./module-01/README.md): Kubernetes architecture trust map
- [Module 02](./module-02/README.md): RBAC review and least-privilege redesign
- [Module 03](./module-03/README.md): Pod hardening to restricted baseline
- [Module 04](./module-04/README.md): Network isolation with NetworkPolicy
- [Module 05](./module-05/README.md): Secrets, exposure vectors, and safer delivery
- [Module 06](./module-06/README.md): Admission control and policy-as-code
- [Module 07](./module-07/README.md): Software supply chain and image trust
- [Module 08](./module-08/README.md): Cluster hardening and platform baselines
- [Module 09](./module-09/README.md): Detection engineering and threat hunting
- [Module 10](./module-10/README.md): Incident response and forensics in Kubernetes
- [Module 11](./module-11/README.md): Governance, multi-tenancy, and platform security operating model
- [Module 12](./module-12/README.md): Cloud provider mappings and reference architecture

## Suggested Usage

- Start with `../MASTER_CLASS_CURRICULUM.md` for the learning sequence.
- Open the corresponding module lab when you reach each curriculum module.
- Keep lab outputs, notes, and screenshots in a separate learner workspace if
  you want this repository to remain a clean teaching baseline.
